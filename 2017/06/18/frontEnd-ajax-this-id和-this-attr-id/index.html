$(this)-id和$(this)-attr('id')

返回元素都是ID，但如果元素没有ID，$(this).id 会返回一个空字符串，而
$(this).attr(“id”) 将返回undefined。

js中：

当中的$(this).attr(‘id’)。”this”是指为”.modifyuser”所在的标签，这个标签当中的属性：id
既$(“.modifyuser”).getAttribute(‘id’)。
